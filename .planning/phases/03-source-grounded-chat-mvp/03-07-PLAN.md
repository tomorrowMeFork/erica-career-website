## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| `.env`/process env→smoke CLI | Local provider configuration enters the script; names may be logged, values must never be logged. |
| CLI args→chat query | User-supplied Korean query enters retrieval and provider prompt construction. |
| Provider errors→terminal | External provider errors can contain tokens or echoed request details and must be redacted. |
| Timeout config→provider option | Local numeric config controls live smoke wait behavior and must fail safely if malformed. |
